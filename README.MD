Week 1 - Activity 3: Power Calculator

This Python program calculates the power of a number (x^y).
The user inputs a base (x) and exponent (y), and the program returns the result.

 How to Run
1. Open terminal
2. Navigate to Week1 folder
3. Run:
   python power_calculator.py

 Example
Input:
2 and 4

Output:
2^4 = 16

 Environment
- Python 3
- VS Code
